### This repository add fileName and Line into Logger
# logger winston filename:line

app using [Express 4](http://expressjs.com/) and [Nodemon global 4](https://www.npmjs.com/package/nodemon).
## Running Locally

```sh
git clone git@github.com:dat0106/logger_winston_filename_line # or clone your own fork
cd logger_winston_filename_line
npm install
npm start
```

Your app should now be running on [localhost:3000](http://localhost:3000/).

## Deploying to Heroku

```
heroku create
git push heroku master
heroku open
```

Alternatively, you can deploy your own copy of the app using the web-based flow:

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)
